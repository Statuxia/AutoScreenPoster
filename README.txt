Как использовать?
1) Нажмите выбранную вами горячую клавишу в окне "Нажмите клавишу" для последующей работы с программой;
2) Нажмите "Подтвердить";
3) При нажатии горячей клавиши в окне ниже через некоторое время появится ссылка на скриншот (хранится 1.5 месяца);
4) "Скопировать" копирует всё окно. "Очистить", соответственно, очищает;

Для чего нужна?
При нажатии заданной клавиши ASP отправляет скриншот сайту и возвращает ответ в виде ссылки.

Если программа работает некорректно, попробуйте запустить от 
имени администратора.
Программа работает ТОЛЬКО на 64 битной системе.
