# AutoScreenPoster

Made by FQGM

-------------

Как пользоваться:

Заходите на VimeWorld;

Нажимаете F2;

Ждете звуковой сигнал.

Получаете ссылку на ваш скриншот.

Программу перезапускать не нужно.
-----
-----------

В py файле нужно ввести свои токены из имгура.
exe файлом можно пользоваться без ввода токена.
